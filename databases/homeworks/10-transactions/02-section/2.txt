При выборе рейса, пользователю отображается самолет с сиденьями
Мы получим список актуальных перелетов для каждого перелета достаем все места которые там есть.
Показываем их пользователю в разных цветах для обозначения статуса (купленные / забронированные / свободные)

BoughtSeats, FreeSeat, CurrentReservedSeats – функции из прошлого дз

Свободные места пользователь может забронировать – Reserve или купить место например как комбинация (Reserve + BuyReserved)
Если пользователь не залогинен можем воспользоваться функцией BuyFree – которая сразу производится покупку.

Если какие-то операции не удалось выполнить выводим сообщение об ошибке, например "Это место занято" +
какую-нибудь кнопку для обновления страницы

Забронированные места залогиненный пользователь можем купить – BuyReserved.

Также может в небольшом окошке показывать статистику по рейсу для залогиненных пользователей (FlightStat)
