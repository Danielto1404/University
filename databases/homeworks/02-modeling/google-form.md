### Сущности

Всего 4 сущности Students, Groups, Courses, Teachers.

```
Students содержит в себе:
+-----------------+----------------------+------------+---------+
| имя атрибута    | описание             | свойства   | домен   |
|-----------------+----------------------+------------+---------|
| student_id      | идентификатор записи | PK, M      | id      |
| student_name    | имя студента         | M          | name    |
| student_surname | фамилия студента     | M          | name    |
+-----------------+----------------------+------------+---------+

Groups содержит в себе:
+----------------+----------------------+------------+------------+
| имя атрибута   | описание             | свойства   | домен      |
|----------------+----------------------+------------+------------|
| group_id       | идентификатор записи | PK, M      | id         |
| group_name     | название группы      | K1, M      | group_name |
+----------------+----------------------+------------+------------+

Courses содержит в себе:
+----------------+-----------------------------------+------------+-------------+
| имя атрибута   | описание                          | свойства   | домен       |
|----------------+-----------------------------------+------------+-------------|
| course_id      | идентификатор записи              | PK, M      | id          |
| course_name    | название курса                    | M          | course_name |
| semester       | семестр в котором проводится курс | O          | semester    |
+----------------+-----------------------------------+------------+-------------+

Teachers содержит в себе:
+-----------------+-----------------------+------------+---------+
| имя атрибута    | описание              | свойства   | домен   |
|-----------------+-----------------------+------------+---------|
| teacher_id      | идентификатор записи  | PK, M      | id      |
| teacher_name    | имя преподавателя     | M          | name    |
| teacher_surname | фамилия преподавателя | M          | name    |
+-----------------+-----------------------+------------+---------+
```

### Связи
```
Сущность Groups связана со Students как one-to-many.
Сущность Groups связана с Courses как many-to-many.
Сущность Courses связана с Teachers как many-to-many.
Courses и Students связаны через ассоциацию Marks как many-to-many.
У Marks есть атрибут mark с доменом mark - оценка.
```

### Домены:
```
id          - int
name        - varchar(50)
group_name  - char(10)
semester    - int
mark        - varchar(2)
course_name - varchar(100)
```

### Преобразование:
```
Сущности раскрылись в таблицы.
Связь Groups - Courses (many-to-many) раскрылась через join table, аналогично раскрылась и связь Courses - Teachers.
Ассоциация Marks раскрылась в сущность Marks (случай аналогичный many-to-many)
с добавлением "нагрузки" в виде оценки.
```
