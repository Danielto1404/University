1. Найдите функциональные зависимости в данном отношении.
2. Найдите все ключи данного отношения.

	+ *GroupId, CourseId*
	+ *StudentId, CourseId*
	+ *StudentId, LecturerId*
4. Найдите неприводимое множество функциональных зависимостей для данного отношения.
